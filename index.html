Proyectos

Mercado Liebre

Desarrollé un ecommerce inspirado en el estilo visual de grandes plataformas de venta online como MercadoLibre. El objetivo era crear una interfaz amigable y familiar para el usuario, que le permitiera explorar una amplia variedad de productos de forma intuitiva. Utilicé HTML y CSS para la estructura y el diseño visual de la página. El diseño responsive se logró mediante el uso de media queries para adaptar la visualización en dispositivos móviles y de escritorio.

Mundo Muebles

En este proyecto diseñamos un ecommerce ficticio enfocándonos solamente en lo visual. Logré que se sintiera acogedor y ordenado. Las imágenes que utilice detallan la calidad de los productos del sitio. También cuenta con categorías en un menú lateral para facilitar la navegación del usuario entre los distintos tipos de muebles, y también incluye algunas ofertas destacadas. Completando el sitio agregué una sección de contacto y un espacio en el cual el usuario puede dejar su mail para recibir ofertas y promociones.
